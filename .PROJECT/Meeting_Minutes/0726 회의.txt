2021-07-26 회의록

* 데이터 분석을 통한 추천 알고리즘 만들기 확정

* 내일까지 각자 흥미 있는 분야에서 추천 알고리즘을 만들만한 데이터 찾아보기
 -> 찾아온 데이터들을 토대로 각자 발표 후 투표로 주제 선정

* 선생님 말씀
 - Readme file에는 제목과 내용을 적으면 되고 내용에는 프로젝트에 대한 설명을 채우면 된다
 - 프로젝트일지 폴더를 만들어서 매일매일이 아니더라도 사건이 있을 때마다 텍스트 파일로 기록해서 프로젝트일지 폴더에 저장하기
 - 프로젝트 폴더 내 팀원들 이름으로 폴더 만들기(완료)

* 회의 중 말·말·말
 - 정서현
 -> 주가 측정 재밌을 거 같다 -> 야후 API로 데이터를 설정해서 하면 데이터 문제 해결 가능할 것 같다

 - 이종원
 -> 주가 측정 -> 다음 주에 몇 퍼센트 이상 오를 종목을 True and False로 알려주는 방식을 생각해봤다

 - 김주원
 -> 알고리즘 형식으로 데이터를 찾아봤다
 -> 이디야 자주 먹는 음료수 -> 바닐라만 마셔 -> 이디야에서 바닐라를 많이 마신 손님들이 다른 카페에서는 어떤 음료를 마실까? -> 하지만 데이터가 많지 않고 있는 데이터들은 해외카페 지점이었다 -> 기각 -> 고객들이 많이 먹는 음료를 토대로 새로운 메뉴를 만드는 생각해봤다
 -> 주식 관련 데이터가 넘치니까 이거로 해도 괜찮지 않을까 생각해봤다

 - 안정모
 -> 주식을 잘 몰라서 딴 거를 찾다가 축구 관련해서 선수들 득점이나 도움에 비례해서 연봉이 얼마나 올라가는지 그런 거 찾아보려고 했는데 선수 개개인 정보라서 연봉이 상세하게 나오지는 않았다. 그래서 그쪽으로 하는 거보다는 일단 데이터양이 중요하니까 그래도 데이터가 제일 많은 주식 쪽으로 하는 게 맞지 않나 싶다.

 - 신준재
 -> 여러 채용공고를 고려했을 때 추천 알고리즘과 관련해서 프로젝트를 하는 것이 모든 팀원이 나중에 취업하는 데에 있어 좋은 포트폴리오가 되지 않을까 생각했다
